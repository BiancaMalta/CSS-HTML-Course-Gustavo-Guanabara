Parágrafos

Para escrever um parágrafo em html, basta colocar tudo no meio de um par de tags <p> e </p>. Se precisar quebrar o texto em algum lugar específico, você pode usar a tag <br>. Mas, caso precise de várias linhas de espaço, é indicado o uso do CSS.

Já para adicionar comentários utilizamos <!--(escreva aqui seu comentário-->, vale ressaltar que eles não apareceram para seu usuário, ao contrário das linhas( <hr> ), dos símbolos( ® © ™ € ¥ ¢ £ δ ↑) e dos emoji( 🖖 🤓).

Testando carga e Imagens

Ao utilizar imagens, é necessário atenção aos direitos autorias. Para isso, abra a ferramenta do google imagens e marque a opção reutilização com modificação ou então use como fonte o site: Unsplash ou Pexels.

Favicon

Favicon é o nome que se dá ao ícone do site, ele é adicionado ao html na área head em forma de link. Existem diversas opções prontas no site iconarchive, mas na hipótese de precisar desenhar o seu ícone, tem-se o site favicon.cc.

Hierarqui de titulos

Os títulos vão de h1 a h6, sendo que cada H é um nível, o qual deve ser seguido rigorosamente. Consequentemente, é necessário a utilização do CSS quando o tamanho não se adequar a parte estética.

Principais formatações

Negrito/ Destaque

Um termo em negrito pode ser escrito com a tag B, a qual não é semântica, ou com a tag strong , que por sua vez é semântica.

Itálico/ Ênfase

Em itálico a tag i é a não semântica e a tag em é a semântica.

Texto marcado

Nos textos marcado temos somente a tag mark ,porém são geradas mais opções ao se colocar o CSS dentro da tag e selecionando o background-color, sendo esta uma formatação pontual.

Texto pequeno

As letras pequenas de um contrato podem ser geradas com a tag small.

Texto deletado/ inserido

A marcação de um texto excluído com a tag del , e a ênfase de um texto inserido com a tag ins .

Texto sobrescrito/subscrito

Textos sobrescritos (como o 20 desse exemplo: x20) são gerados com a tag sup, por sua vez textos subscritos (como o 2 desse exemplo: H2O) são gerados com a tag sub.

Código-fonte

Ao se inserir códigos, a tag code possibilita um melhor espaçamento.

document.getElementById('teste')

Entretanto, para se ter identação, é necessário envelopar a tag code com a tag pre. 

      
num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'O número {num} é PAR')
else:
    print(f'O número {num} é ÍMPAR)

Listas

Listas Ordenadas

Usamos a tag ol para abrir uma lista e cada item deve ficar dentro da tag li, como no exemplo abaixo:

  1. acordar
  2. tomar cafe
  3. escovar os dentes

Podemos personalizar a formatação da lista colocando type dentro da tag de ol.

Listas não ordenadas

São listas que não precisam ser enumeradas, para isso abrimos a tag ul.

  • pao
  • leite
  • tomate

Assim como na anterior ela tbm possui diversos types.

Lista de definição

Já para lista de definições abrimos a tag dl, utilizando dt para o nome de cada elemento que será explicado e dd para a explicação.

Trabalhando com links:

Externos

A tag a possibilita adicionar um link e para que esse link ao ser aberto não se sobreponha à sua pagina, utilize: target="_blank" rel="external".

Voce pode acessar o meu repositório publico no Github e conferir essa usabilidade.

Internos

Links internos geram coneções com outras páginas que também estão dentro do nosso servidor. Supondo que a página atual seja a primeira página do site e que você queira acessar a segunda página presente nesse servidor. Valeu lembra que ./ é na mesma pag e ../ é na página anteiror.

Para download

A ferramente do usuário clicar em algo no seu site e automaticamente começar o download é resultado de um pdf baixado e compactado dentro de uma tag a, juntamente com: download="meulivro.zip" type="application/zip"

Mídia

Imagem dinâmica

Seu site precisa se adaptar a diferentes dispositivos, será necesssário disponibilizar diferentes sources, os quais, somados com a tag img, vão ser envelopados pela tag picture. A organização dessa estrutura deve seguir uma order de tamanho, com a menor no primeiro source e a maior na tag img, a limitação de tela deve utilizar a tag max-with.

Áudio

A inserção de áudio ocorre pela tag audio + controls para controles + autoplay para tocar assim que abrir a página. A tag source também pode ser usada aqui, possibilitando varias opções de carregamento ao usuário.

Vídeo

Se tratando de vídeo, não compensa disponibilizar a melhor resolução. Cabe lembrar que vídeos hospedados diretamente do seu servidor vão consumir muito tráfego de dados, a melhor opção é adicionar vídeos de serviços de hospedagem de vídeo, como o YouTube, basta o iframe dispobinilizado.

CSS

Estilos Inline

Ao se colocar dentro de cada tag um style, são gerados estilos locais, que em excesso poluí seu html, logo um metódo para configurações pontuais.

Estilos Internos

Os estilos internos são baseado em seletores e criados dentro da área de head. Exemplo:

body{ background-color: lightsteelblue; font-family: Arial, Helvetica, sans-serif; font-size: 20px; }

A desvantagem dessa técnica é que além de ocupar muito mais espaço que o html, ela se limita a apenas aquela página.

Estilos Externos

A apliacação do style ficará em outra página, para isso, dentro do head colocamos um link para o css. Não se esqueça que a compatibilidade com a acentuação, depende de @charset "UTF-8";.
